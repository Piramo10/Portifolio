Gerenciador de Finanças

Estou trabalhando neste projeto. Nele o Objetivo é ajudar a manter o controle das finanças pessoais. O usuário poderá cadastrar suas receitas e despesas, categorizá-las e visualizar gráficos de fluxo de caixa e saldo mensal, e tudo isso com uma integração de Inteligencia Artificial para ajudar o usuario. O backend será desenvolvido com Node.js e Express, utilizando um banco de dados MongoDB para armazenamento dos dados financeiros. O frontend será construído com JavaScript vanilla, proporcionando uma interface simples e intuitiva para facilitar o gerenciamento das finanças pessoais.
